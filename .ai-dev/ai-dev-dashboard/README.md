# AI Dev Dashboard

> **Version:** 1.6.0 | **Status:** 🚧 In Development

A full-stack project management dashboard for the AI Dev Agency documentation system. Built with **NestJS** (backend) and **Vite + React 18** (frontend), it provides a visual interface for tracking epics, stories, bugs, sprints, bursts, and browsing project documentation — all powered by markdown files and YAML frontmatter.

## Architecture

```
.ai-dev-dashboard/
├── backend/                 # NestJS 10 + TypeScript (port 3002)
│   ├── src/
│   │   ├── board/           # Board module (kanban data from board.json)
│   │   ├── bugs/            # Bug tracking module
│   │   ├── bursts/          # Burst planning module
│   │   ├── cli/             # CLI integration module
│   │   ├── common/          # Shared services (MarkdownService)
│   │   ├── documents/       # Documentation viewer (read-only)
│   │   ├── epics/           # Epic management module
│   │   ├── health/          # Health check endpoint
│   │   ├── progress/        # Progress tracking module
│   │   ├── sprints/         # Sprint planning module
│   │   ├── stories/         # Story management module
│   │   └── data/            # board.json (generated by /sync-board)
│   └── test/                # E2E tests
├── frontend/                # Vite + React 18 + TypeScript + Tailwind CSS (port 3001)
│   ├── src/
│   │   ├── components/      # Reusable components (board, layout, docs)
│   │   ├── pages/           # Dashboard, Sprints, Bursts, Bugs, Docs, CLI
│   │   ├── services/        # API clients (typed, per-domain)
│   │   └── store/           # Zustand stores (board, sprint)
│   └── e2e/                 # Playwright E2E tests
└── ai-dev-dashboard.sh      # Launcher script
```

## Quick Start

### Prerequisites

- Node.js 18+
- npm 9+

### 1. Generate Board Data

```bash
# In the project root, run the /sync-board skill to generate board.json
/sync-board
```

### 2. Start Backend

```bash
cd .ai-dev-dashboard/backend
npm install
npm run build
npm start
# Backend running at http://localhost:3002
# Swagger docs at http://localhost:3002/api/swagger
```

### 3. Start Frontend

```bash
cd .ai-dev-dashboard/frontend
npm install
npm run dev
# Frontend running at http://localhost:3001
```

### Using the Launcher Script

```bash
bash .ai-dev-dashboard/ai-dev-dashboard.sh start    # Start both services
bash .ai-dev-dashboard/ai-dev-dashboard.sh stop     # Stop services
bash .ai-dev-dashboard/ai-dev-dashboard.sh restart  # Restart services
```

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/health` | Health check |
| `GET` | `/api/board` | Full board data |
| `GET` | `/api/stories` | Stories (filter: `?version=`, `?status=`) |
| `GET` | `/api/releases` | Release history |
| `GET` | `/api/metrics` | Metrics + suggestions |
| `GET` | `/api/epics` | All epics |
| `GET` | `/api/bugs` | All bugs |
| `GET` | `/api/sprints` | All sprints |
| `GET` | `/api/sprints/active` | Active sprint |
| `GET` | `/api/bursts` | All bursts |
| `GET` | `/api/bursts/active` | Active burst |
| `GET` | `/api/docs` | All documents |
| `GET` | `/api/docs/search?q=` | Search documents |
| `GET` | `/api/docs/:path` | Document by path |
| `POST` | `/api/cli/execute` | Execute CLI command |
| `GET` | `/api/cli/history` | Command history |
| `GET` | `/api/progress` | Progress stats |

## Frontend Pages

| Page | Route | Description |
|------|-------|-------------|
| **Dashboard** | `/` | Kanban board with drag-and-drop, filters, metrics |
| **Sprints** | `/sprints` | Sprint planning, burndown/velocity charts |
| **Bursts** | `/bursts` | Burst session tracking, flow state |
| **Bugs** | `/bugs` | Bug tracking with severity badges |
| **Docs** | `/docs` | Confluence-like documentation viewer with Mermaid.js |
| **Terminal** | `/cli` | CLI command execution with history |

## Tech Stack

### Backend
- **Framework:** NestJS 10
- **Language:** TypeScript
- **Documentation:** Swagger/OpenAPI at `/api/swagger`
- **Markdown:** gray-matter for YAML frontmatter parsing

### Frontend
- **Build:** Vite
- **Framework:** React 18
- **Language:** TypeScript
- **Styling:** Tailwind CSS + @tailwindcss/typography
- **State:** Zustand (persisted) + React Query
- **Drag & Drop:** @dnd-kit
- **Charts:** Recharts
- **Markdown:** react-markdown + rehype-highlight + Mermaid.js
- **Icons:** lucide-react
- **Testing:** Playwright (E2E), Vitest (unit)
